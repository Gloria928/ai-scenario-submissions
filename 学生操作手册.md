# AI场景编程作业提交系统 - 学生操作手册

## 🎯 欢迎使用AI场景编程作业提交系统！

### **系统简介**
这是一个专为AI场景编程课程设计的智能作业提交系统，帮助你：
- 📝 轻松提交编程作业
- ⚡ 获得即时自动评分
- 📊 查看详细评分报告
- 🔄 随时查询提交状态

---

## 🚀 快速开始

### **第一步：访问系统**
**推荐方式**：通过Gitee教材页面访问
```
在Gitee教材页面找到"作业提交"链接或iframe嵌入的表单
```

**备用方式**：直接访问测试页面
```
https://ai-scenario-submissions.onrender.com/test-submit.html
```

### **第二步：首次使用配置**
1. **打开页面**：看到系统测试界面
2. **测试连接**：点击"测试健康检查"按钮
3. **确认连接**：看到"✅ 健康检查测试成功！"

### **第三步：提交作业**
1. **点击**："测试提交作业"按钮
2. **等待**：系统自动提交测试作业
3. **保存**：复制显示的"提交ID"

---

## 📝 正式提交作业

### **方法一：使用测试页面提交**
1. **修改测试代码**：在测试页面修改代码内容
2. **填写信息**：
   - 姓名：你的真实姓名
   - 学号：你的学号（可选）
   - 作业题目：选择对应的作业
   - 编程语言：选择Python（默认）
   - 代码：粘贴或输入你的代码

3. **点击提交**：等待系统处理

### **方法二：使用API直接提交（高级）**
如果你熟悉API调用，可以直接使用：

```bash
curl -X POST "https://ai-scenario-submissions.onrender.com/api/submit" \
  -H "Content-Type: application/json" \
  -d '{
    "studentName": "你的姓名",
    "studentId": "你的学号",
    "assignment": "作业题目",
    "language": "python",
    "code": "你的代码"
  }'
```

### **提交要求**
- ✅ **代码长度**：不超过5000字符
- ✅ **编程语言**：支持Python、JavaScript、Java
- ✅ **提交频率**：每15分钟最多10次
- ✅ **必填信息**：姓名、作业题目、代码

---

## 🔍 查询提交状态

### **方法一：在测试页面查询**
1. **复制提交ID**：提交成功后显示的ID
2. **粘贴到查询框**：在"查询状态"输入框
3. **点击查询**：查看详细状态

### **方法二：使用API查询**
```bash
curl "https://ai-scenario-submissions.onrender.com/api/status/你的提交ID"
```

### **状态说明**
- **pending**：等待处理（刚提交）
- **submitted**：已提交到GitHub
- **graded**：评分完成
- **failed**：提交失败

### **预计时间**
- ⏰ **处理时间**：1-3分钟
- 📊 **评分时间**：1-3分钟
- 🕒 **总耗时**：2-6分钟

---

## 📊 查看评分结果

### **方法一：在测试页面查看**
提交状态显示"graded"后，页面会显示：
- 🏆 分数（0-100分）
- 📋 详细评分报告
- 💡 改进建议

### **方法二：查看GitHub Issue**
1. **访问**：`https://github.com/Gloria928/ai-scenario-submissions/issues`
2. **找到**：你的作业Issue（标题包含你的姓名）
3. **查看**：Issue评论中的评分结果

### **方法三：使用提交ID直接访问**
```
https://ai-scenario-submissions.onrender.com/api/status/你的提交ID
```

### **评分标准**
- 🎯 **功能正确性**：70分（通过所有测试用例）
- 📝 **代码质量**：30分（注释、命名、结构）
- 🏆 **附加分**：10分（创意、效率）

## ⚠️ 常见问题

### **1. 提交失败怎么办？**
**可能原因**：
- 网络连接问题
- 代码超过5000字符
- 15分钟内提交过于频繁

**解决方案**：
1. 检查网络连接
2. 简化代码或分多次提交
3. 等待15分钟后重试
4. 联系教师获取帮助

### **2. 找不到提交ID怎么办？**
**找回方法**：
1. 检查浏览器历史记录
2. 查看GitHub Issues（按姓名搜索）
3. 联系教师查询提交记录

**预防措施**：
- ✅ 立即复制保存提交ID
- ✅ 发送到自己的邮箱或笔记
- ✅ 截图保存提交成功页面

### **3. 评分结果不满意怎么办？**
**改进步骤**：
1. 仔细阅读评分报告
2. 理解错误原因
3. 修改代码重新提交
4. 对比前后评分差异

**注意事项**：
- 🔄 可以多次提交同一作业
- 📈 系统记录所有提交历史
- 🎯 取最高分作为最终成绩

### **4. 系统无法访问怎么办？**
**临时方案**：
1. 等待几分钟后重试
2. 使用备用测试页面
3. 直接联系教师提交

**原因说明**：
- 🌙 Render免费服务可能休眠
- 🔧 系统维护升级
- 🌐 网络临时问题
